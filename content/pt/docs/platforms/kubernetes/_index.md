[Kubernetes](https://kubernetes.io/) é um sistema open source para implantação
automatizada, escalonamento e gerenciamento de aplicações em contêineres.
Tornou- se uma ferramenta amplamente adotada pela indústria, levando a uma maior
necessidade de ferramentas de observabilidade. Em resposta, o OpenTelemetry
criou muitas ferramentas diferentes para ajudar os usuários do Kubernetes a
observar seus clusters e serviços.
