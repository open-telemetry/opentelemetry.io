OpenTelemetry Collector складається з одного бінарного файлу, який ви можете використовувати по-різному для різних випадків використання. У цьому розділі описуються загальні шаблони розгортання, їхні випадки використання разом з перевагами та недоліками, а також найкращі практики для конфігурацій колектора для крос-середовищних та мульти-бекендних розгортань. Для розгляду питань безпеки розгортання дивіться [Найкращі практики хостингу колектора][security].
