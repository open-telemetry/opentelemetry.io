Javaでゼロコード計装を行う一般的なオプションには、Java エージェント JAR、Spring Boot Starter、Quarkus OpenTelemetry Extension があります。
サービスやアプリケーションコードを手動で計装する方法については、[手動計装](/docs/languages/java/instrumentation/)を参照してください。
