OpenTelemetry [instrumentación automática][auto instrumentation] es compatible
con los idiomas que se enumeran a continuación.

Si usa Kubernetes, puedes usar el [Operador OpenTelemetry para
Kubernetes][otel-op] para [inyectar instrumentación
automática][inject auto instrumentation] para .NET, Java, Node.js, Python o Go
en su aplicación.
