La [demostración de OpenTelemetry](/docs/demo/) hace el siguiente:

- Proporciona un ejemplo realista de un sistema distribuido que puede utilizarse
  para demostrar la instrumentación y la observabilidad de OpenTelemetry.
- Crea una base para que proveedores, creadores de herramientas y otros amplíen
  y demuestren sus integraciones con OpenTelemetry.
- Crea un ejemplo real para que los colaboradores de OpenTelemetry lo utilicen
  para probar nuevas versiones de la API, el SDK y otros componentes o mejoras.

Si se pregunta:

- ¿Cómo debo usar el SDK para mi lenguaje?
- ¿Cuál es la mejor manera de usar las API de OpenTelemetry?
- ¿Cómo debo configurar mis servicios?
- ¿Cómo debo configurar mi recopilador de OpenTelemetry?
- ¿Cómo considero la [arquitectura](/docs/demo/architecture/) de un sistema
  usando OpenTelemetry?
