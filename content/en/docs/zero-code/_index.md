OpenTelemetry [auto instrumentation][] is supported for the languages listed
below in the section index.

If you are using Kubernetes, you can use the [OpenTelemetry Operator for
Kubernetes][otel-op] to [inject auto instrumentation][] for .NET, Java, Node.js,
Python, or Go into your application.
